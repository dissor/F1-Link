&par Example Description 

The example shows how to use RCT to generate second interrupt.
download program and then reset(power off  to power on). 
LED2 will blink every second which is generated by RCT interrupt.
  
&par Directory contents 

  - RTC/Second/src/apm32e10x_it.c     Interrupt handlers
  - RTC/Second/src/main.c                      Main program

&par Hardware and Software environment

  - This example runs on APM32E103_MINI Devices.
