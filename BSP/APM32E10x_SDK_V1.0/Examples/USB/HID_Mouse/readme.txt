&par Example Description 

This example shows how to use USB. Dwonload the program, then connect the
device to your computer through USB. If the USB is configed correctly, you can 
see a new HID-mouse in your computer.

&par Hardware and Software environment

  - This example runs on APM32E103 MINI Devices.
