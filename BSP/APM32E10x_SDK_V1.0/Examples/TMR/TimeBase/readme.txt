&par Example Description 
 
This example shows how to realize timing one second by using TMR1 peripheral  generating time base .
LED2 will toggle pre second.

&par Directory contents 

  - TMR/TimeBase/src/apm32e10x_int.c           Interrupt handlers
  - TMR/TimeBase/src/main.c                           Main program


&par Hardware and Software environment

  - This example runs on APM32E103_MINI Devices.
